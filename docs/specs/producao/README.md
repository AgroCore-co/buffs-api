# PRODUCAO - Indice de Especificacoes

Este diretorio contem as regras do modulo de producao.

Documentos:

- 00-producao-visao-geral.md
- 01-producao-lactacao-ordenha.md

Escopo coberto:

- Estrutura do modulo raiz e subdominios de lactacao, ordenha, estoque diario, retirada, laticinios e predicao.
- Regras de negocio de ciclos de lactacao, controle leiteiro e alertas de producao.
- Fluxo operacional de consolidacao de estoque, coleta por laticinio e gestao de industrias.
- Contratos de entrada e saida (DTOs), validacoes de dados e limites de campos.
- Integracao com recursos compartilhados do Core (auth, logger, database, cache, paginacao e validadores).
